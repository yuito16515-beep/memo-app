# メモ検索アプリ

## 概要
メモの追加・編集・削除に加えて、キーワード検索や並び替えができるメモ管理アプリです。  
localStorage を利用してデータを保存しているため、ページを再読み込みしてもメモが保持されます。

## 使用技術
- HTML
- CSS
- JavaScript
- localStorage

## 実装機能
- メモの追加（Create）
- メモの編集（Update）
- メモの削除（Delete）
- キーワード検索
- 新しい順 / 古い順の並び替え
- データの保存（localStorage）

## 工夫した点
- filter を用いた検索機能
- 配列の並び替えによる表示順の変更
- id を用いたデータ管理で安全な削除処理
- シンプルで見やすいUI設計
