# メモ検索アプリ

## 概要
JavaScriptとlocalStorageを用いて開発した、メモ管理用のWebアプリです。
ブラウザ上でメモの保存・検索・編集・削除ができます。

## 公開URL
（ここにGitHub PagesのURLを記載）

## 主な機能
- メモの追加・表示・削除
- メモの編集機能
- キーワード検索機能
- 並び替え機能（新しい順・古い順）
- データの永続化（localStorage）

## 使用技術
- HTML
- CSS
- JavaScript（Vanilla JS）
- Web Storage API（localStorage）

## 制作背景
JavaScriptの基礎学習の成果を形に残すため、
実用的なアプリとしてメモ管理ツールを制作しました。

## 工夫した点
- CRUD機能を意識した設計にしたこと
- filterを用いたリアルタイム検索処理
- UIが直感的になるようレイアウトを調整したこと
- アクセシビリティを意識し、labelを適切に設定したこと

## 苦労した点
HTMLとJavaScriptのID不一致による不具合の修正に苦労しましたが、
開発者ツールを活用して原因を特定し、解決しました。
